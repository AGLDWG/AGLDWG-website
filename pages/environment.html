environment.data.gov.au

This page is the home, or root, page of the environment.data.gov.au persistent IRI namespace.

environment.data.gov.au is a legacy a persistent IRI domain maintained by the Australian Government Linked Data Working Group which contains a few IRI patterns implemented in 2016 - 2019 but for which no new patterns will be accepted.

Note that this domain uses HTTP, not HTTPS.

All existing patterns are:

http://environment.data.gov.au/defenvironment.data.gov.au Definitions
http://environment.data.gov.au/def/opObservable-Properties ontology
http://environment.data.gov.au/def/ba/glossaryBioregional Assessment Programme Glossary
http://environment.data.gov.au/def/ba/landscape-classificationBioregional Assessment Programme landscape classification
http://environment.data.gov.au/def/ba/licenceA list of licences used in the Bioregional Assessment Programme glossary
http://environment.data.gov.au/def/ba/sourceA list of sources used in the Bioregional Assessment Programme glossary
http://environment.data.gov.au/def/propertyA collection of observable properties
http://environment.data.gov.au/def/featureGeneric features used in definitions of observable properties
http://environment.data.gov.au/def/objectA set of substances and taxa which may appear as the subject (identified object) of some observable properties.
http://environment.data.gov.au/def/quantity-kindA set of quantity kind individuals that are used in water-quality observations but are not included in generic QUDT
http://environment.data.gov.au/def/unitSome units of measure used in groundwater, surface water and marine water quality observations

The patterns above were all created by CSIRO as part of either the eReefs or Bioregional Assessments projects.

All new persistent IRIs issued by the AGLDWG will use the linked.data.gov.au domain only.
